# 💻 Harith Haikal's Personal Tech Blog

A personal blog website built with HTML, CSS, and JavaScript as part of the CSD34203 Special Topics in Software Development course at UniSZA.

---

## 📖 Description

This is a fully responsive personal blog page where I share my thoughts, experiences, and knowledge about technology and programming. The blog reflects my journey as a computing student and my passion for the digital world.

---

## ✨ Features

- 🏠 **Home Page** — Hero section with featured blog post cards
- 👤 **About Page** — Personal background, skills, and timeline
- 📝 **Blog Page** — 3 full blog posts on tech topics
- 📬 **Contact Page** — Contact form with JavaScript validation
- 📱 **Responsive Design** — Works on mobile and desktop
- 🎨 **Dark Theme** — Modern dark UI with smooth animations
- ⚡ **Scroll Animations** — Elements animate into view on scroll

---

## 🛠 Technologies Used

| Technology | Purpose |
|-----------|---------|
| HTML5 | Page structure and content |
| CSS3 | Styling, layout, animations |
| JavaScript | Interactivity, form validation, scroll effects |
| Google Fonts | Typography (Syne + Space Mono) |
| Git & GitHub | Version control and hosting |

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/YOUR-USERNAME/personal-blog.git
   ```

2. Open the project folder:
   ```bash
   cd personal-blog
   ```

3. Open `index.html` in your browser — **that's it!**

> No frameworks or installations needed. Pure HTML/CSS/JS.
